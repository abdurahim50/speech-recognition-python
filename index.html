Building a Simple Speech Recognition Tool in Python

Overview

In this project, we’ll use Python’s speech_recognition library to:

  • Capture audio input from a microphone.
  • Process the audio and convert it to text using the Google Web Speech API.
  • Handle common errors like timeouts or unrecognizable speech.

Prerequisites

1. Python Installation

Ensure Python 3.7 or above is installed on your machine. Verify this by running:

python --version

If not installed, download Python here.

2. Install Required Libraries

Install the following libraries:

pip install SpeechRecognition
pip install pyaudio

Note: If you face issues installing pyaudio, download a precompiled binary for your Python version here.

Step 1: Writing the Code

Here’s the complete script for our speech recognition tool:

import speech_recognition as sr

# Initialize the Recognizer
recognizer = sr.Recognizer()

# Capture audio from the microphone
with sr.Microphone() as source:
    print("Speak please...")
    try:
        # Listen for speech with a timeout of 10 seconds
        audio = recognizer.listen(source, timeout=10)
        print("Recording stopped.")
        
        # Recognize speech using Google's Web Speech API
        text = recognizer.recognize_google(audio)
        print("You said:", text)
    except sr.WaitTimeoutError:
        print("No speech detected within the timeout period.")
    except sr.UnknownValueError:
        print("Sorry, I could not understand the audio.")
    except sr.RequestError as e:
        print(f"Could not request results from Google Speech Recognition service; {e}")

Step 2: Running the Script

1. Save the script as speech_recognition.py.

2. Run it using:

python speech_recognition.py

Expected Output

When you speak, the script transcribes your speech. For example:

Speak please...
Recording stopped.
You said: Hello world, this is a test!

If no speech is detected:

No speech detected within the timeout period.

Step 3: Troubleshooting Common Issues

1. Microphone Not Detected

Ensure your microphone is connected and enabled. Check system settings for the default input device.

2. Installation Errors

For pyaudio installation issues:

On Debian-based systems:

sudo apt-get install portaudio19-dev
pip install pyaudio

On macOS:

brew install portaudio
pip install pyaudio

3. Speech Not Recognized

Reduce background noise and speak clearly into the microphone.

Step 4: Enhancing the Project

Expand your tool by:

  • Adding a GUI: Use libraries like Tkinter or PyQt for a graphical interface.
  • Integrating with APIs: Utilize the recognized text for chatbots, search engines, or home automation systems.
  • Supporting Multiple Languages: Specify a language code in recognize_google (e.g., language='es-ES' for Spanish).
